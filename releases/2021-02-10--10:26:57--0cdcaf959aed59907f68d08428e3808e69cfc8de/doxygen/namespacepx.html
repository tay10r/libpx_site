◆ BlendMode

enum px::BlendMode
strong

Describes how two colors are combined.

Enumerator
Normal 

The normal blend mode applies simple alpha compositing to the image. It is usually the default and expected behavior.

Subtract 

This is used to subtract color from the resultant image. It can be used for erasing contents of the image as well as removing certain color components.

◆ openDoc()

int px::openDoc ( Document *  doc,
const char *  filename,
ErrorList **  errList = nullptr 
)

Imports data from an external document.

Parameters
docA pointer to a document returned from createDoc
filenameThe path to the file to import the data from.
errListAn optional parameter to store the error list at. See Error List API for more information. If a pointer is passed to this parameter, then it is set to null until there are actual parser errors to assign. Therefore, the pointer passed to this function may still be null depending on whether or not syntax errors were found.
Returns
If the document was opened properly, then zero is returned. On error, the value of errno is returned. If the document failed to open because of a syntax error, then EINVAL is returned. In either case, errList is only created if there's syntax erros. So users should null-check the pointer before calling any of the functions in pxErrorApi
