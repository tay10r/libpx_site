Getting Started

To get started using this library, the first set of functions to use are one of:

  • px::createDoc
  • px::createImage

Checkout Document API and Image API for more information on using those objects.

Here's a code example that demonstrates the initialization and cleanup routines.

#include <libpx.hpp>
int main() {
/* The document gets rendered to this object. */
px::Image* img = px::createImage();
/* Used to describe what to render (geometry, color, etc.) */
px::Document* doc = px::createDoc();
/* Use library */
/* Finish using library and release memory */
px::closeDoc(doc);
px::closeImage(img);
return 0;
}

Adding Content

Once an instance of px::Document has been created, content can be added to it so that we have something to render.

Rendering Content

After content has been added to our document, we can render it onto an instance of px::Image.

This can be done using one of the px::render functions.
